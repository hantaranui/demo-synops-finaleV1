# SynOps - Prototype final

Prototype narratif realise pour le hackathon Women in Gen AI 2026.

SynOps illustre un assistant generatif de mediation semantique pour transformer des signaux heterogenes en information exploitable, tracable et validee par un humain.

## Contenu

- ouverture immersive en poste avance isole ;
- traitement tactique frugal et local ;
- IA ciblee sur les ambiguities ;
- validation humaine dans Grist ;
- remontee multi-niveaux ;
- redescente strategique ;
- securite et mode degrade ;
- conclusion et affiche finale de remerciements.

## Lancer le prototype

Le prototype est un site statique.

Ouvrir directement `index.html` dans un navigateur, ou lancer un serveur local depuis ce dossier.

```bash
python3 -m http.server 4173
```

Puis ouvrir :

```text
http://127.0.0.1:4173/index.html
```

## Donnees

Les donnees et situations affichees sont fictives et non sensibles.
